# Operator Guide

## First Deployment
1. Register operator, generators/routes, waste classifications, staff
   and robots.
2. Import existing manifest and chain-of-custody history.
3. Run read-only classification-scope and collection/transport robot
   mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run chain-of-custody report and audit export.

## Minimum Production Controls
- waste-classification-scope validation before any collection/transport
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (unmanifested
  transport, out-of-permit treatment)
- evidence-backed chain-of-custody reports
- audit export for every dispatch, sign-off and report
- backup manual manifest process

## Certification
Certified operators must prove robot-safety integrity, manifest
discipline, evidence-backed chain-of-custody reporting and human review
for treatment/transport-affecting actions.
